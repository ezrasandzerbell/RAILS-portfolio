# _Demo Portfolio_

#### _Performs API call to Github and returns project names as hyperlinks_

#### By _**Ezra Sandzer-Bell**_

Configuration:

* $ postgres
* $ bundle
* $ Rails db:create
* $ Rails db:seed
* $ Rails db:migrate
* $ Rails db:test:prepare
* $ Rails serve
* $ rspec
